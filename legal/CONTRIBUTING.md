## Requirements

1. **SPDX Headers:** Include SPDX license identifiers in all files:
   - Nix files: `# SPDX-License-Identifier: Apache-2.0`
   - Shell scripts: `# SPDX-License-Identifier: Apache-2.0` (after shebang)
   - Markdown files: `<!-- SPDX-License-Identifier: Apache-2.0 -->`

2. **Code Quality:** Test changes thoroughly before submitting

3. **Documentation:** Update relevant documentation with your changes

4. **Commit Messages:** Write clear, descriptive commit messages
